# **App Name**: PropSource AI

## Core Features:

- Demand Capture: Capture property demand details: Property ID (auto-generated), date/time stamp, geo-location (current location capture and map display), size, floor, readiness to occupy, site type, safety features, ceiling height, and commercial terms like rent and deposit.
- Intelligent Matching: Use AI to match property demands with available properties. An AI-powered tool will analyze property features and demand criteria to suggest suitable matches.
- Demand Circulation: Circulate property demands to registered users: Property developers, agents, and property owners can receive notifications about matching property requirements.
- Property Submission: Users submit property details that match circulated demands, including user type, contact information, statutory approvals, electricity and fire safety details, docks, and additional information.
- Image Uploads: Allow users to upload images of the property

## Style Guidelines:

- Primary color: HSL(210, 60%, 50%) - A moderately saturated blue evokes trust and professionalism, but steers clear of corporate clichés (#2979FF).
- Background color: HSL(210, 20%, 95%) - A light, desaturated blue provides a clean, neutral backdrop, ensuring readability and a modern feel (#F2F8FF).
- Accent color: HSL(180, 70%, 40%) - A vivid teal complements the primary blue, drawing attention to key actions and interactive elements (#1AB2B2).
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Clean and structured layout with clear sections for each data category (Property, Commercials, User, Statutory Approvals, etc.).
- Consistent and professional iconography to represent property features, approval statuses, and other key information.
